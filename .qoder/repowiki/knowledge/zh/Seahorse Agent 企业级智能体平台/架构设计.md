- **前后端分离架构**：前端（React）通过 RESTful API 和 SSE/WebSocket 流式接口与后端通信，实现智能对话与管理功能的实时响应。
- **Clean Architecture 核心**：后端采用六边形架构，`seahorse-agent-kernel` 定义领域模型与端口接口，`seahorse-agent-adapter-*` 模块实现具体技术适配（如 JDBC、Milvus、OpenAI），确保业务逻辑与技术细节解耦。
- **插件化与自动配置**：基于 Spring Boot Starter 机制，通过 `seahorse-agent-spring-boot-starter` 统一装配内核、适配器及定时任务，支持动态加载 Agent Skill 与 MCP 工具。
- **统一运维与治理**：通过 `docker-compose` 编排全栈环境，结合 Aegis 治理中心规范，实现从代码基线、数据库迁移到监控告警的全生命周期管理。
