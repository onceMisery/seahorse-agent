- **视图层**：`ChatPage.tsx` 作为主容器，集成 `react-resizable-panels` 实现聊天区与工作区（WorkspaceInspector）的动态分栏布局。
- **状态管理层**：采用 Zustand + Immer 构建全局状态。`chatStore` 负责会话生命周期、消息流式追加及 Agent 事件归一化；`workbenchStore` 管理侧边栏检查器的显隐与标签切换。
- **网络通信层**：`useStreamResponse.ts` 封装了健壮的 SSE 客户端，内置看门狗超时机制、指数退避重试逻辑以及断点续传（Resume）支持。
- **渲染优化层**：引入 `RenderBuffer` 类配合 `requestAnimationFrame` 对高频流式文本进行节流渲染，确保 UI 流畅性。
- **数据适配层**：`chatStreamUtils.ts` 提供统一的 Agent 事件解析器，将后端多样化的 SSE 事件（如 Timeline, Artifacts, Approvals）转换为前端标准化的类型结构。
