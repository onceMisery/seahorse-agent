- **数据持久化层**：以 PostgreSQL 为核心，通过 `seahorse_init.sql` 定义基础表结构（用户、会话、知识库），并利用 Flyway 风格的版本化迁移脚本（`V2__` 至 `V13__`）演进多租户隔离（RLS）、计费、审计及 Agent 市场等模块。
- **向量存储集成**：启用 `pgvector` 扩展并定义 `t_knowledge_vector` 表，结合 Milvus 独立部署配置，构建混合检索架构。
- **部署编排层**：在 `resources/docker/` 下提供 Milvus 与 Pulsar 的 Docker Compose 配置，区分标准版与轻量级（内存受限）环境，支持本地开发与生产隔离。
- **运维与验证工具**：通过 `scripts/` 目录下的 JS 和 PowerShell 脚本，实现后端 API 映射提取与自动化接口健康检查，形成闭环的基础设施验证流程。
- **知识资产库**：在 `resources/docs/knowledge/` 中结构化存储业务规范（保险、OA）与内部制度（HR、IT），作为 RAG 系统的原始语料来源。
