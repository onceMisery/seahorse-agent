- **轻量级部署**：在内存受限环境下，使用 `docker compose -f resources/docker/lightweight/milvus-stack-2.6.6.compose.yaml up -d` 启动带有内存限制的 Milvus 栈。
- **API 验证**：运行 `powershell scripts/verify-backend-apis.ps1` 对本地后端服务（默认 localhost:9090）进行全量接口健康检查。
- **前端映射生成**：执行 `node scripts/extract-backend-mappings.js` 从 Java 源码中提取注解并生成前端可用的 `backendEndpointManifest.ts`。
