# 开源项目设计亮点落地方案

### bolt.new Workbench → Workspace Inspector

参考来源：`D:\code\docs\stackblitz-bolt.new\12-workbench-and-editor.md`

落地方式：
- `/chat` 页面改为左侧对话 + 右侧 Workspace Inspector 分屏布局

未引入：WebContainer、终端、文件系统、完整 IDE

### CopilotKit A2UI → A2UI-lite 白名单渲染器

参考来源：`D:\code\docs\CopilotKit-CopilotKit\21-a2ui-declarative-rendering.md`

落地方式：
- 仅支持 5 种安全组件：`metric`、`table`、`source_grid`、`callout`、`action_row`
- 版本校验：`version === "seahorse-a2ui-lite/v1"`
- 不使用 `dangerouslySetInnerHTML`，不执行脚本
- MIME 类型：`application/vnd.seahorse.a2ui+json`

未引入：完整 CopilotKit 运行时、任意 React 组件渲染

### CopilotKit Web Inspector → Admin Agent Inspector

参考来源：`D:\code\docs\CopilotKit-CopilotKit\26-web-inspector-debugging.md`

落地方式：
- 新增 `/admin/agent-inspector` 路由，支持 `/:runId` 参数
- 4 个调试视图：Events（原始事件流）、State（运行状态）、Context（来源/产物/记忆）、Tools（工具调用/审批）
- 后端新增 `GET /api/agent-runs/{runId}/events?afterSeq=0` 接口
- 受 `AI_INFRA_CONSOLE` 特性门控保护
