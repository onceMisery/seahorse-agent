- 识别问题类型
→ 判断适用地区/场景/对象
→ 先查强制标准
→ 再查内部 SOP
→ 再查经验教训
→ 比对冲突
→ 输出结论和不确定点
  - 回答约束格式：
  - 结论：可以 / 不建议 / 禁止 / 信息不足 
  - 依据：引用哪些条款、页码、文档版本
  - 适用条件：在什么场景下成立
  - 风险等级：高 / 中 / 低
  - 改进建议：具体怎么做
  - 不确定点：还缺哪些现场信息
